# aws-cdk-templates-showcase

Showcase of AWS CDK templates.

- [Unique Domain Hosting](https://github.com/dayjournal/aws-cdk-templates-showcase/tree/main/unique-domain-hosting)  

<br>

## Usage

Execution environment

- node v16.10.0
- npm v7.24.0
- aws-cdk v2.15.0

<br>

Install package

```bash
npm install
```

<br>

deploy

```bash
cdk deploy
```
